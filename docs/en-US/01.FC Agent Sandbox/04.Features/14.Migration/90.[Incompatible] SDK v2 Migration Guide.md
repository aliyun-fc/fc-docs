# [Incompatible] SDK v2 Migration Guide

The E2B SDK v2 migration guide is E2B's own SDK upgrade documentation and does not belong to FC Agent Sandbox feature documentation.

FC Agent Sandbox documentation should not replicate E2B's SDK migration guide. When using FC Agent Sandbox, developers should select the SDK version compatible with the current FC Agent Sandbox and refer to [Usage Constraints](../../01.Getting%20Started/04.Usage%20Constraints.md) and [E2B Compatibility](../../01.Getting%20Started/05.E2B%20Compatibility.md).
