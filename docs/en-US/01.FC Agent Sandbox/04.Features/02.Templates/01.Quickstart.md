# Quickstart

A template defines the base runtime environment for a sandbox. By preinstalling system dependencies, language runtimes, toolchains, and application code in a template, you avoid reinstalling them after every sandbox is created.

This topic walks through one full flow: define a template, build it, and create a sandbox from it. For the concepts behind this flow, see [How It Works](03.How%20It%20Works.md).

## Prerequisites

- An E2B-compatible image reachable over VPC. You can use an official image from [Build Custom Image Templates](04.Build%20Custom%20Image%20Templates.md) to try the flow first.
- The connection parameters `E2B_API_KEY`, `E2B_API_URL`, and `E2B_DOMAIN`. See [E2B SDK Connection Parameters](../../07.Developer%20Reference/02.E2B%20SDK%20Connection%20Parameters.md). The SDK reads these environment variables automatically, so the examples below do not pass them into method calls.

Install the SDK and load the environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install e2b==2.31.0 python-dotenv
```

```bash
# .env — replace with your own values
E2B_API_KEY=e2b_xxx
FROM_IMAGE="fc-e2b-registry.cn-beijing.cr.aliyuncs.com/runtime/code-interpreter-v1:v0.0.31"
E2B_API_URL=https://api.cn-beijing.e2b.fc.aliyuncs.com
E2B_DOMAIN=cn-beijing.e2b.fc.aliyuncs.com
```

## Build a template and create a sandbox

```python
import os
import time

from dotenv import load_dotenv
from e2b import Sandbox, Template, default_build_logger

load_dotenv()

# The SDK reads E2B_API_KEY / E2B_API_URL / E2B_DOMAIN automatically.
# 1. Define the template from an image and build it.
build = Template.build(
    Template().from_image(os.environ["FROM_IMAGE"]),
    name=f"template-{int(time.time())}",
    cpu_count=2,
    memory_mb=2048,
    on_build_logs=default_build_logger(),
)

# 2. Create a sandbox from the template that was just built.
sandbox = Sandbox.create(template=build.template_id, timeout=900)
print(f"template_id: {build.template_id}")
print(f"sandbox_id: {sandbox.sandbox_id}")
sandbox.kill()
```

```typescript
import { Sandbox, Template, defaultBuildLogger } from "e2b";

// The SDK reads E2B_API_KEY / E2B_API_URL / E2B_DOMAIN automatically.
// 1. Define the template from an image and build it.
const build = await Template.build(
  Template().fromImage(process.env.FROM_IMAGE),
  `template-${Date.now()}`,
  { cpuCount: 2, memoryMB: 2048, onBuildLogs: defaultBuildLogger() },
);

// 2. Create a sandbox from the template that was just built.
const sandbox = await Sandbox.create(build.templateId, { timeoutMs: 900_000 });
console.log(`template_id: ${build.templateId}`);
console.log(`sandbox_id: ${sandbox.sandboxId}`);
await sandbox.kill();
```
