| **Feature** | **Change Type** | **Description** | **Related Documentation** |
| --- | --- | --- | --- |
| FC Agent Sandbox | General Availability | FC Agent Sandbox is generally available in the China North 2 (Beijing) region. It provides isolated cloud execution environments that you can create, run, connect to, and release on demand for AI agent workloads, code execution, command execution, file processing, and temporary service preview scenarios.
