# 阿里云 SDK & API 说明

本文汇总云沙箱原生 OpenAPI、阿里云 SDK 和阿里云 CLI 的调用入口。云沙箱 OpenAPI 提供 `Team`、`API Key`、`Quota` 等资源管理能力，这些能力属于阿里云 OpenAPI / POP SDK 控制面能力，不是 E2B SDK 兼容 API。

如果您使用 E2B SDK 或 E2B CLI 接入云沙箱，请参见 [E2B SDK 兼容 API 清单](./01.E2B%20SDK%20兼容%20API%20清单.md) 和 [E2B SDK 接入参数说明](./02.E2B%20SDK%20接入参数说明.md)。

## 调用前准备

调用云沙箱 OpenAPI 前，请先完成以下准备：

- 开通函数计算和云沙箱相关能力。
- 准备可用的阿里云账号或 RAM 用户，并授予调用云沙箱 OpenAPI 所需权限。
- 确认调用地域。OpenAPI、SDK、CLI 调用时使用的地域应与云沙箱资源所在地域一致。
- 配置阿里云访问凭证。生产环境建议使用 RAM 用户、RAM 角色或其他临时凭证方案，不要在代码中明文写入 AccessKey。

## 服务接入点

最新服务接入点请以 [云沙箱服务接入点](https://api.aliyun.com/product/FCSandbox?tab=endpoints) 为准。

## 查看 OpenAPI 文档

云沙箱 OpenAPI 文档提供接口列表、请求参数、响应结构、错误信息和在线调试能力。开发前建议先通过 OpenAPI 文档确认目标接口的参数要求和返回结构。

查看文档：[云沙箱 OpenAPI 文档](https://api.aliyun.com/document/FCSandbox/2026-05-09/overview)

## 通过阿里云 SDK 调用云沙箱服务

在业务系统中调用云沙箱服务时，建议使用阿里云 SDK。SDK 页面提供安装方式、初始化客户端示例和接口调用示例，并支持切换 Java、Python、Go、TypeScript 等语言。

查看文档：[云沙箱 SDK 安装与调用示例](https://api.aliyun.com/api-tools/sdk/FCSandbox?version=2026-05-09&language=java-async-tea&tab=primer-doc)

## 通过阿里云 CLI 调用云沙箱服务

阿里云 CLI 适合本地验证、脚本化调用和排障。使用前请先安装阿里云 CLI，并完成凭证、地域等基础配置。

阿里云 CLI 安装和配置方法请参见：[阿里云 CLI 使用指南](https://help.aliyun.com/zh/cli/)

查看 CLI 调用示例：[使用阿里云 CLI 调用云沙箱服务](https://api.aliyun.com/api-tools/cli/FCSandbox/2026-05-09)

## 使用建议

- 如果需要确认接口是否存在、参数如何填写，优先查看 OpenAPI 文档。
- 如果需要在应用代码中稳定调用云沙箱服务，优先使用阿里云 SDK。
- 如果需要快速验证接口或编写轻量运维脚本，可以使用阿里云 CLI。
