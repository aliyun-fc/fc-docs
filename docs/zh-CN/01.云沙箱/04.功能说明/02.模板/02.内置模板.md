# 内置模板

内置模板由云沙箱平台提供和维护，用于快速创建标准 Sandbox。当前可直接使用的内置模板覆盖基础命令与文件访问、代码执行场景。

如果您只是需要 Python、Node.js、文件管理或代码执行等通用能力，建议优先使用内置模板。只有当内置模板无法满足业务依赖、系统库、运行时版本或企业标准化要求时，再创建自定义镜像模板。

## 模板列表

| 模板 | 适用场景 | 说明 |
| --- | --- | --- |
| [base 模板](02.内置模板/01.base-模板.md) | 基础命令、文件访问、SDK 连通性验证 | 默认基础模板，不提供 Code Interpreter 服务 |
| [code-interpreter-v1 模板](02.内置模板/02.code-interpreter-v1-模板.md) | AI Agent 代码执行、数据分析、文件处理 | 快速入门和代码解释器示例默认使用 `code-interpreter-v1` |

> **说明**
>
> `base` 与 `code-interpreter-v1` 在账号开通云沙箱后会自动就绪，可直接用模板名创建沙箱。

## 查看模板

使用 CLI 查看当前账号可用模板：

```bash
export E2B_API_KEY="<your-api-key>"
export E2B_API_URL="https://api.cn-beijing.e2b.fc.aliyuncs.com"
export E2B_DOMAIN="cn-beijing.e2b.fc.aliyuncs.com"

e2b template list
```

## 使用模板创建 Sandbox

使用 `e2b_code_interpreter` SDK 时，默认创建 `code-interpreter-v1` 沙箱，通常不需要显式指定模板。

使用通用 `e2b` SDK 时，可通过 `template` 参数显式选择模板。SDK 会自动读取 `E2B_API_KEY`、`E2B_API_URL`、`E2B_DOMAIN` 环境变量。Python 示例：

```python
from e2b import Sandbox

sandbox = Sandbox.create(template="code-interpreter-v1")
```

TypeScript 示例：

```typescript
import { Sandbox } from "e2b";

const sandbox = await Sandbox.create("code-interpreter-v1");
```
